# Borikua Smart Contracts 🇵🇷

A collection of smart contracts celebrating **Puerto Rican** culture, **Boricua** resistance, and **Taíno** heritage on Base blockchain. Built to honor the indigenous **Taíno** people of **Borikén** (Puerto Rico) and promote solidarity in the **Caribbean** diaspora.

## ✨ Features

### HelloWorld Contract
- **Multilingual support**: English, Spanish, and Taíno greetings
- **Cultural pride**: "Yo soy Borikua, pa' que tú lo sepas\!"
- **Owner functionality**: Updateable messages
- **Event logging**: Message update tracking

### PRFlag Contract  
- **ASCII Art Flag**: Black Puerto Rican resistance flag
- **Cultural messaging**: Resistance and solidarity themes
- **Community interaction**: Public message posting
- **Event tracking**: Flag display and message logging

## 🛠 Quick Start

### Prerequisites
- [Foundry](https://book.getfoundry.sh/getting-started/installation)
- [Git](https://git-scm.com/)

### Installation
```bash
git clone https://github.com/yourusername/borikua-contracts.git
cd borikua-contracts
forge install
```

### Testing
```bash
forge test -vv
```

## 🌟 Cultural Significance

### The Black Flag
The black Puerto Rican flag represents:
- **Resistance** against colonial oppression
- **Solidarity** in social justice movements  
- **Cultural pride** and Boricua identity

### Language Preservation
- **Taíno/Taino**: Honoring indigenous **Borikén** heritage with "Taiguey" (good day)
- **Spanish**: Celebrating **Puerto Rican** linguistic identity  
- **English**: Bridging **Caribbean** and diaspora communities
- **Cultural preservation**: Keeping **Boricua** traditions alive on blockchain
